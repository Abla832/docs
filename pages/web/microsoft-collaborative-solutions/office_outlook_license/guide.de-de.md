## Ziel

Sie möchten eine Outlook-Lizenz nutzen, um Ihre E-Mails auf Ihrem Exchange Account einzusehen und zu verwalten.

OVHcloud bietet Ihnen den Outlook-E-Mail-Client zum Preis von 2 Euro ohne MwSt. pro Monat an. Es wird einem Exchange Account von OVHcloud zugewiesen, dessen Verwaltung Sie innehaben.

Nach Ihrer Bestellung können Sie Outlook aus einer der 6 folgenden Versionen herunterladen:

- Outlook 2013 für Windows 32 Bit.
- Outlook 2013 für Windows 64 Bit.
- Outlook 2016 für Windows 32 Bit.
- Outlook 2016 für Windows 64 Bit.
- Outlook 2011 für 64 Bit MAC.
- Outlook 2016 für 64 Bit MAC.

**In dieser Anleitung erfahren Sie, wie Sie eine Outlook-Lizenz von Ihrer OVHcloud Exchange-Plattform aus bestellen und installieren.**

## In der praktischen Anwendung

### Outlook-Lizenz bestellen

#### Für einen Hosted Exchange Account

Verbinden Sie sich mit Ihrem [OVHcloud Kundencenter](https://www.ovh.com/auth/?action=gotomanager&from=https://www.ovh.de/&ovhSubsidiary=de) und wählen Sie dann den Bereich `Web Cloud`. Klicken Sie links im Menü auf `Microsoft`{.action}, dann auf `Exchange`{.action} und wählen Sie die betreffende **Hosted Exchange** Plattform aus.

Klicken Sie auf das Icon `...`{.action} rechts neben dem betreffenden Account und dann auf `Outlook-Lizenz bestellen`{.action}.

![Outlook](images/order-outlook01.png){.thumbnail}

Legen Sie das Verlängerungsintervall Ihrer Outlook Lizenz fest und bestätigen Sie die Bedingungen. Klicken Sie anschließend auf `Weiter`{.action}.

![Outlook](images/order-outlook02.png){.thumbnail}

Verwenden Sie nach der Zusammenfassung Ihrer Bestellung den Button `Bezahlen`{.action}, um den Bestellschein zu erstellen. Sie werden auf eine neue Seite weitergeleitet, auf der Sie Ihre Bestellung mit den Zahlungsmitteln begleichen können.

Warten Sie einen Moment, bis Ihre Outlook-Lizenz in Ihrem Kundencenter verfügbar ist.

#### Für einen Private Exchange Account

Verbinden Sie sich mit Ihrem [OVHcloud Kundencenter](https://www.ovh.com/auth/?action=gotomanager&from=https://www.ovh.de/&ovhSubsidiary=de) und wählen Sie dann den Bereich `Web Cloud`. Klicken Sie links im Menü auf `Microsoft`{.action}, dann auf `Exchange`{.action} und wählen Sie die betreffende **Private Exchange** Plattform aus.

Klicken Sie auf das Icon `...`{.action} rechts neben dem betreffenden Account und dann auf `Outlook-Lizenz aktivieren`{.action}.

![Outlook](images/order-outlook03.png){.thumbnail}

In einem Fenster werden Sie gefragt, ob Sie die Outlook-Lizenz für Ihren Exchange Account aktivieren möchten. Klicken Sie auf `Bestätigen`{.action}, um die Lizenz für den betreffenden Account zu aktivieren.

![Outlook](images/order-outlook04.png){.thumbnail}

Warten Sie einen Moment, bis Ihre Outlook-Lizenz in Ihrem Kundencenter verfügbar ist.

> [!primary]
>
> Die Abrechnung Ihrer Outlook Lizenz wird jeden Monat in die Abrechnung Ihrer Private Exchange Plattform integriert.
>

### Outlook herunterladen und installieren

Laden Sie nun die Installationsdatei von Outlook für Ihre Maschine herunter.

Klicken Sie im Verwaltungsinterface Ihrer Exchange Plattform auf das Icon `...`{.action} rechts neben dem betreffenden Konto und dann auf `Outlook Lizenz verwenden`{.action}

Wählen Sie die Version im Drop-down-Menü abhängig von Ihrem Betriebssystem und der Sprache aus und klicken Sie dann auf `Weiter`{.action}

![Outlook](images/order-outlook05.png){.thumbnail}

Nach einigen Augenblicken wird ein Download-Link sowie ein Lizenz-Key zur Verwendung bei der Installation auf Ihrem Server erstellt.

![Outlook](images/order-outlook06.png){.thumbnail}

Die heruntergeladene Datei ist im .ISO-Format, also einem Festplattenimage. Starten Sie die Installation und geben Sie den Lizenz-Key ein, wenn dieser angefordert wird.

### Outlook-Lizenz von Ihrem Account löschen

Verbinden Sie sich mit Ihrem [OVHcloud Kundencenter](https://www.ovh.com/auth/?action=gotomanager&from=https://www.ovh.de/&ovhSubsidiary=de) und wählen Sie dann den Bereich `Web Cloud`. Klicken Sie links im Menü auf `Microsoft`{.action}, dann auf `Exchange`{.action} und wählen Sie die entsprechende Exchange-Plattform aus.

Klicken Sie auf das Icon `...`{.action} rechts neben dem betreffenden Account und dann auf `Outlook-Lizenz löschen`{.action}

![Outlook](images/order-outlook07.png){.thumbnail}

Nach der Validierung wird die Lizenz zum Ablaufdatum endgültig gelöscht
